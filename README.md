# Loan-Approval-Analysis
End-to-end Data Analytics project including EDA using Python, SQL data analysis, and interactive Power BI dashboards.
This project focuses on analyzing loan approval patterns using data analytics techniques. 
The analysis examines important factors such as Debt-to-Income (DTI) ratio, average income, total income, and loan amount to understand their impact on loan approvals. 
The project includes data cleaning, exploratory data analysis (EDA), SQL queries, and visualization. The goal is to identify key trends and insights that influence loan approval decisions. 
Tools used include Python, SQL, and Power BI.
